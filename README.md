# AURORA-GUI

Локальная панель для поиска публично опубликованных связей номера телефона.

## Что делает

- нормализует номер и определяет оператора/регион;
- ищет открытые страницы;
- отбрасывает каталоги «кто звонил» и неподтверждённые совпадения;
- извлекает публичные ФИО, организации, email, профили, объявления и публичные адреса;
- сохраняет итог в `findings.html`, `findings.txt` и `findings.json`;
- сохраняет сырую выдачу отдельно в `raw_web_results.json`.

## Установка поверх существующей версии

В Kali выполни:

```bash
cd ~
git clone https://github.com/DrummerIke/AURORA-GUI.git AURORA-GUI-new
bash AURORA-GUI-new/install.sh
```

Установщик создаёт резервную копию текущей папки `~/AURORA-GUI`, если она ещё не была подключена к GitHub.

## Запуск

```bash
cd ~/AURORA-GUI
python3 app.py
```

Открой:

```text
http://127.0.0.1:8080
```

## Обновление

```bash
cd ~/AURORA-GUI
git pull
python3 -m pip install -r requirements.txt
```

## Ограничения

Проект работает только с открыто опубликованными данными. Совпадение номера с именем, профилем или адресом не доказывает текущего владельца. Публичный адрес может относиться к офису, месту оказания услуги или объявлению.
