# AURORA-GUI

Локальная исследовательская панель для поиска и сопоставления публично опубликованных связей по номеру телефона, email и username.

## Режимы

### Phone Intelligence

- нормализация номера;
- оператор, страна и регион;
- поиск точных публичных упоминаний;
- извлечение ФИО, организаций, email, профилей, объявлений и публичных адресов;
- каскадное обогащение найденных email и username.

### Email Intelligence

- поиск открытых публикаций с точным email;
- Holehe для публичных регистрационных признаков;
- извлечение username из локальной части email;
- дополнительный поиск через Sherlock и Maigret.

### Username Intelligence

- Sherlock;
- Maigret;
- открытые публикации и профили в поисковой выдаче.

## Состояние модулей

Главная страница показывает, какие инструменты доступны. Машиночитаемый статус:

```text
http://127.0.0.1:8080/modules
```

Базовые компоненты устанавливаются обязательно. Sherlock, Maigret и Holehe устанавливаются как дополнительные модули: сбой одного из них не блокирует запуск Aurora.

## Установка поверх существующей версии

В Kali или Ubuntu:

```bash
cd ~
git clone https://github.com/DrummerIke/AURORA-GUI.git AURORA-GUI-new
bash AURORA-GUI-new/install.sh
```

Установщик создаёт резервную копию текущей папки `~/AURORA-GUI`, если она ещё не была подключена к GitHub.

## Запуск

```bash
cd ~/AURORA-GUI
./run.sh
```

Открой:

```text
http://127.0.0.1:8080
```

## Обновление

```bash
cd ~/AURORA-GUI
git pull
bash install.sh
```

## Результаты

Каждое расследование создаёт отдельную локальную папку в `cases/`. Эта папка исключена из Git и не публикуется. В ней формируются HTML- и JSON-отчёты, а также технические результаты внешних модулей.

## Ограничения

Проект работает только с открыто опубликованными данными. Совпадение номера, email или username с именем, профилем, адресом или организацией не доказывает личность человека. Любая связь требует ручной проверки по первоисточнику.
